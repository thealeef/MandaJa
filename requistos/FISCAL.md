# Fiscal do MandaJa

## Objetivo

Estruturar a base operacional para cumprimento de requisitos fiscais relacionados a estabelecimentos, pagamentos e documentos.

## Requisitos iniciais

- Armazenar dados fiscais do estabelecimento de forma segura.
- Apoiar a emissao de dados para tributacao e integracao com sistemas fiscais, quando aplicavel.
- Separar dados fiscais de dados operacionais sempre que possivel.
- Registrar o historico de alteracoes cadastrais e documentos enviados.

## Campos previstos

- CNPJ ou CPF
- Inscricao estadual ou municipal
- Regime tributario
- Endereco fiscal
- Email fiscal
- Documentos legais e comprovantes
