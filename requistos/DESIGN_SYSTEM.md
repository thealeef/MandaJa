# Design system do MandaJa

## Objetivo

Estabelecer a base visual para os ambientes web e mobile do MandaJa, mantendo identidade consistente e experiencia amigavel.

## Identidade visual

- Azul petroleo: #0F4C5C
- Laranja de acao: #FF9F1C
- Verde entrega: #2ECC71
- Cinza gelo: #F5F7FA
- Grafite: #1F2937
- Vermelho erro: #E63946
- Amarelo aviso: #FACC15

## Diretrizes de interface

- Visual moderno, limpio e profissional.
- Uso de espacos generosos e hierarquia clara.
- Botões e ações com destaque visual.
- Componentes responsivos para mobile e desktop.
- Status de pedido com cores consistentes.

## Componentes previstos

- Botões primarios e secundários
- Cards de produto e pedido
- Badges de status
- Modais de confirmacao
- Formularios de cadastro e login
- Listas de entregas e notificações
