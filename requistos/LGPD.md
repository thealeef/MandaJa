# LGPD do MandaJa

## Objetivo

Definir a base para o tratamento adequado de dados pessoais e o cumprimento das exigencias da Lei Geral de Protecao de Dados.

## Requisitos iniciais

- Coletar apenas dados necessarios para a prestacao do servico.
- Informar de forma clara o motivo da coleta.
- Garantir consentimento explicito para termos, politicas e notificacoes.
- Permitir exclusao, atualizacao e acesso aos dados do usuario.
- Registrar eventos de tratamento sensível e acessos administrativos.

## Itens prioritarios

- Termos de uso e politica de privacidade
- Consentimentos de notificacoes e localizacao
- Controle de acesso a dados pessoais
- Retencao de dados conforme regra de negocio
- Processos para solicitação de exclusao e portabilidade
