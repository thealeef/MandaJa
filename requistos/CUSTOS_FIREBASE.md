# Custos do Firebase do MandaJa

## Objetivo

Mapear os principais custos operacionais esperados ao usar a stack Firebase no projeto MandaJa.

## Principais componentes

- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Cloud Functions
- Firebase Hosting
- Firebase Cloud Messaging
- App Check

## Consideracoes financeiras

- O custo pode crescer com volume de pedidos, uso de storage e chamadas de funcoes.
- A estrutura de dados deve ser otimizada para evitar leituras e escritas desnecessarias.
- O uso de regras e indexacao adequada reduz custo operacional.
- Devem ser definidos limites, alertas e monitoramento de consumo.

## Recomendacao inicial

Criar um plano de custos desde o inicio, com estimativas para desenvolvimento, homologacao e producao, e revisar periodicamente o consumo real.
